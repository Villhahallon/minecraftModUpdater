## minecraftModUpdater
 Simple minecraft Mod updater using modrinth API

I created this to learn python and to update my minecraft mods in a better way.

AI was used to get me going.



# Basic guide:

 - Create a mods.txt file where you can find it.
 - add mods to this file
 - copy the name from the url, Fabric api as en example ```fabric-api```
   
 <img width="412" height="55" alt="image" src="https://github.com/user-attachments/assets/31d0c216-980d-4aae-99f9-6d0365a5f81b" />
 
 - One mod per line
 - run the application
 - select minecraft version. (Only tested with 1.21.1, 1.21.11 currently)
 - select loader (Fabric, Forge, Quilt)
 - select the mod.txt file.
 - run updater with the button.

A mods folder will be created in the same directory as the exe file.
It should create a subfolder with the selectet minecraft version.
Copy mods to the Minecraft mod folder.

The application will update the mods if run again. ( Only to the created mods folder)
